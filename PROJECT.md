# FedorBot — План разработки асинхронного Telegram-бота на Python

## Общая идея

FedorBot — это интерактивная энциклопедия пословиц и поговорок, которая позволяет пользователям:

- Просматривать коллекцию народных мудростей
- Получать интерпретацию значения пословиц от различных ИИ-моделей (GigaChat, OpenAI и др.)
- Сравнивать разные интерпретации одной и той же пословицы
- Добавлять новые пословицы в базу данных
- Делиться интересными интерпретациями с другими пользователями

Бот использует асинхронную архитектуру на Python с библиотекой aiogram для взаимодействия с Telegram API и SQLAlchemy для хранения данных в базе.

**FedorBot** — асинхронный Telegram-бот на Python с использованием библиотеки aiogram и SQLAlchemy для работы с базой данных. Бот является энциклопедией пословиц и поговорок, предоставляя пользователям возможность интерпретировать их значение с помощью различных моделей искусственного интеллекта.

### Ключевые принципы

- **Telegram-платформа** — бот работает в Telegram, используя API через библиотеку aiogram
- **Асинхронная архитектура** — использование async/await для эффективной обработки множества пользователей
- **Локальная база данных** — для хранения сообщений используется SQLite (fedorbot.db) с возможностью миграции на PostgreSQL
- **Python 3.10+** — основной язык разработки с использованием современных возможностей языка
- **SQLAlchemy ORM** — для типобезопасной работы с базой данных
- **Переменные окружения** — конфигурация через .env файл с использованием python-dotenv
- **Сборка и запуск** — проект запускается напрямую как Python-скрипт

## Итоговая архитектура

```
[Пользователь]
     ↓
[Telegram] ←→ [FedorBot (aiogram)]
                    ↓
             [SQLAlchemy ORM] → [SQLite/PostgreSQL (fedorbot.db)]
                    ↓
             [ИИ-модули] → [GigaChat / OpenAI / другие]
```

- **Основные зависимости:**
  - aiogram — для работы с Telegram API
  - sqlalchemy — для работы с базой данных
  - aiosqlite — асинхронный драйвер SQLite
  - python-dotenv — управление переменными окружения
  - pydantic — валидация данных

- **Скрипты запуска:**
  - `python src/bot.py` — запуск бота
  - `python -m pytest` — запуск тестов
