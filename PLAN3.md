# План реализации этапа 3: Модуль ИИ и интерпретация пословиц

## Обзор этапа

Этап 3 направлен на реализацию функционала интерпретации пословиц с помощью различных ИИ-моделей. Бот должен позволять пользователям получать и сравнивать интерпретации пословиц от разных моделей ИИ.

## Аналитика текущего состояния

На основе анализа кодовой базы, выявлено, что большая часть функционала уже реализована:

1. **Сеть и интеграция ИИ** (`src/network.py`):
   - Реализована поддержка PolzaAI, GigaChat, Yandex GPT и OpenAI-совместимых API
   - Есть обработка ошибок, таймаутов, различных статусов ответов
   - Механизм выбора модели на основе данных из БД

2. **Обработка пословиц** (`src/handlers/proverb_handlers.py`):
   - Есть обработчик `process_analyze_proverb` для анализа пословиц
   - Реализована отправка запросов ко всем активным моделям из БД
   - Сохранение ответов в таблицу `ai_responses`
   - Отображение результатов пользователю

3. **Конфигурация и промпты**:
   - Промпт для ИИ хранится в `prompt.txt`
   - Поддержка переменных окружения через `src/config.py`
   - Модели ИИ хранятся в базе данных с возможностью активации/деактивации
