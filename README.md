# FedorBot — Асинхронный Telegram-бот-энциклопедия пословиц с ИИ-анализом

FedorBot — это интерактивная энциклопедия народных пословиц и поговорок, которая позволяет пользователям не только изучать мудрость народов, но и получать её современную интерпретацию от различных моделей искусственного интеллекта.

## 🚀 Основные возможности

- 📚 База пословиц с возможностью просмотра и добавления
- 🤖 Интерпретация пословиц с помощью ИИ (PolzaAI, GigaChat, Yandex GPT и др.)
- 📨 Ответы от разных моделей приходят **отдельными сообщениями по мере готовности**
- 🔐 Администрирование с правами администратора
- 💬 Интерактивный интерфейс с клавиатурой и кнопками
- 🧠 Хранение истории анализа в БД (`ai_responses`)
- 🔐 Параметры моделей ИИ в отдельной таблице (поддержка промптов на модель)

## 🌐 Web-интерфейс админки

FedorBot поддерживает **веб-админку**, позволяющую управлять содержимым бота через удобный графический интерфейс в браузере.

### 🚀 Возможности веб-интерфейса

- 🔐 **Безопасный вход по паролю** (настраивается в `.env`)
- 📊 **Просмотр и фильтрация** всех пословиц, промтов и моделей ИИ
- ➕ **Добавление новых пословиц и промтов** без перезапуска бота
- 🗑 **Удаление** нежелательных записей
- ⚙️ **Управление активностью моделей ИИ** (вкл/выкл)
- 🔍 **Поиск** по тексту пословиц и промтов
- 💬 **Вкладка «Анализы ИИ»** — просмотр ответов моделей по пословицам
- 📋 **Копирование** и **скачивание** ответов в виде текстового файла
- 📈 **API с Swagger-документацией** для интеграций

### 🖥 Как открыть

1. Запустите веб-сервер:

   ```powershell
   python web_app.py
   ```

2. Откройте в браузере: <http://localhost:5000>

3. Войдите с логином (ваше имя из Telegram) и паролем из `.env`.

#### 🔐 Настройка доступа

Добавьте в файл `.env`:

- `ADMIN_PASSWORD=ваш_надёжный_пароль`
- `SECRET_KEY=секретный_ключ_для_сессий`

🔒 Пароль хранится только в `.env`, не коммитьте его в репозиторий!

### Деплой веб-админки на Vercel

Развернуть только веб-интерфейс на Vercel можно по инструкции в [DEPLOY_VERCEL.md](DEPLOY_VERCEL.md). Там же — про облачную БД и то, что сам бот на Vercel не запускается.

## 🛠 Архитектура и технологии

- Flask — лёгкий веб-фреймворк
- Flask-RESTX — API с автоматической Swagger-документацией

- SQLAlchemy — работа с SQLite через ORM
- Jinja2 (в шаблонах) — динамическая генерация HTML
- Bootstrap-подобный CSS — чистый и понятный интерфейс

🧩 Особенности

Админка не требует отдельного сервера — работает локально.
Все изменения автоматически синхронизируются с основной базой данных (fedorbot.db).
Подходит для быстрого редактирования контента и диагностики.

💡 Совет: Используйте веб-интерфейс для массового добавления пословиц или тестирования промтов перед запуском анализа в Telegram.

- **Python 3.10+** — основной язык
- **aiogram 3.x** — асинхронный фреймворк для Telegram-ботов
- **SQLAlchemy ORM** — работа с базой данных
- **SQLite** — локальная база данных (с возможностью миграции на PostgreSQL)
- **aiohttp** — асинхронные HTTP-запросы к API моделей ИИ
- **python-dotenv** — управление переменными окружения
- **Pydantic** — валидация данных

### Схема базы данных

- `users` — информация о пользователях
- `messages` — переписка с ботом
- `proverbs` — пословицы и поговорки
- `prompts` — промпты для ИИ-анализа (привязка к моделям)
- `models` — конфигурация моделей ИИ (название, провайдер, api_key_var, api_url)
- `ai_responses` — история ответов от моделей ИИ
- `comparisons` — сравнения интерпретаций (схема для будущего функционала)

## 📦 Установка и запуск

### 1. Клонирование репозитория

```powershell
git clone https://github.com/username/fedorbot.git
cd fedorbot
```

### 2. Настройка виртуального окружения

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

### 3. Установка зависимостей

```powershell
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта (образец: `.env.example`):

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=123456789,987654321

DATABASE_URL=sqlite+aiosqlite:///./fedorbot.db

# PolzaAI
POLZAAI_API_KEY=ваш_ключ

# GigaChat (Сбер)
GIGACHAT_CLIENT_ID=ваш_client_id
GIGACHAT_CLIENT_SECRET=ваш_client_secret

# Yandex GPT
YANDEX_OAUTH_TOKEN=ваш_oauth_токен
YANDEX_FOLDER_ID=ваш_folder_id

# Веб-админка
ADMIN_PASSWORD=ваш_пароль
SECRET_KEY=секретный_ключ_сессий
```

### 5. Запуск бота

Из **корня проекта**:

```powershell
python src/bot.py
```

или

```powershell
python -m src.bot
```

> Запуск **без IDE и без Docker** — пошагово в [RUN_WITHOUT_IDE.md](RUN_WITHOUT_IDE.md).

## 🧩 Команды и интерфейс бота

### Команды

- `/start` — начать работу
- `/help` — справка по командам

### Интерфейс (кнопки)

**Для всех пользователей:**
- Выбор пословицы — список с пагинацией (статус 🤖N / ⏳)
- Кнопка **«Анализ ИИ»** — запуск интерпретации (ответы приходят по мере готовности)
- Кнопка **«Поделиться»** — поделиться пословицей

**Для администраторов** (кнопка «Меню управления»):
- **Пословица** — подменю: Добавить, Удалить
- **Анализ ИИ** — выбор пословицы для анализа
- **Просмотр анализа ИИ** — просмотр сохранённых ответов моделей, кнопка «Анализировать заново»
- **Модели** — вкл/выкл моделей ИИ
- **Промты** — добавление, удаление, привязка промпта к модели

## 📦 Сборка установщика (Inno Setup)

1. Установите [Inno Setup 6](https://jrsoftware.org/isinfo.php).
2. Обновите версию из `version.py` и сгенерируйте define для установщика:
   ```powershell
   python scripts/get_install_version.py
   ```
3. Соберите установщик (из корня проекта):
   ```powershell
   "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer\fedorbot.iss
   ```
   Готовый файл: `installer\output\FedorBot-Setup-<версия>.exe`.
