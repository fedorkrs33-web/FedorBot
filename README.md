# FedorBot — Асинхронный Telegram-бот-энциклопедия пословиц с ИИ-анализом

FedorBot — это интерактивная энциклопедия народных пословиц и поговорок, которая позволяет пользователям не только изучать мудрость народов, но и получать её современную интерпретацию от различных моделей искусственного интеллекта.

## 🚀 Основные возможности

- 📚 База пословиц с возможностью просмотра и добавления
- 🤖 Интерпретация значения пословиц с помощью ИИ (PolzaAI, GigaChat, Yandex GPT и др.)
- ⚖️ Сравнение интерпретаций от разных моделей
- 🔐 Администрирование с правами администратора
- 📊 Статистика использования
- 💬 Интерактивный интерфейс с клавиатурой и кнопками
- 🧠 Хранение истории анализа и кэширование популярных ответов
- 🔐 Параметры моделей ИИ хранятся в отдельной таблице базы данных

## 🌐 Web-интерфейс админки

Начиная с версии 0.5, FedorBot поддерживает **веб-админку**, позволяющую управлять содержимым бота через удобный графический интерфейс в браузере.

### 🚀 Возможности веб-интерфейса

- 🔐 **Безопасный вход по паролю** (настраивается в `.env`)
- 📊 **Просмотр и фильтрация** всех пословиц, промтов и моделей ИИ
- ➕ **Добавление новых пословиц и промтов** без перезапуска бота
- 🗑 **Удаление** нежелательных записей
- ⚙️ **Управление активностью моделей ИИ** (вкл/выкл)
- 🔍 **Поиск** по тексту пословиц и промтов
- 📈 **API с Swagger-документацией** для интеграций
- 💬 **Просмотр ответов ИИ** к конкретной пословице

### 🖥 Как открыть

1. Запустите веб-сервер:

   ```bash
   python web_app.py

1. Откройте в браузере:
<http://localhost:5000>

2. Войдите с логином (ваше имя из Telegram) и паролем из .env
🔐 Настройка доступа
Добавьте в файл .env:

 ADMIN_PASSWORD=ваш_надёжный_пароль
 SECRET_KEY=секретный_ключ_для_сессий

🔒 Пароль хранится только в .env, не коммитьте его в репозиторий!

## 🛠 Архитектура и технологии

- Flask — лёгкий веб-фреймворк
- Flask-RESTX — API с автоматической Swagger-документацией

- SQLAlchemy — работа с SQLite через ORM
- Jinja2 (в шаблонах) — динамическая генерация HTML
- Bootstrap-подобный CSS — чистый и понятный интерфейс

🧩 Особенности

Админка не требует отдельного сервера — работает локально.
Все изменения автоматически синхронизируются с основной базой данных (fedorbot.db).
Подходит для быстрого редактирования контента и диагностики.

💡 Совет: Используйте веб-интерфейс для массового добавления пословиц или тестирования промтов перед запуском анализа в Telegram.

- **Python 3.10+** — основной язык
- **aiogram 3.x** — асинхронный фреймворк для Telegram-ботов
- **SQLAlchemy ORM** — работа с базой данных
- **SQLite** — локальная база данных (с возможностью миграции на PostgreSQL)
- **aiohttp** — асинхронные HTTP-запросы к API моделей ИИ
- **python-dotenv** — управление переменными окружения
- **Pydantic** — валидация данных

## 📦 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/username/fedorbot.git
```

### 2. Настройка виртуального окружения

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.\.venv\Scripts\activate  # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=

# Ключи для ИИ-моделей
GIGACHAT_API_KEY=ваш_ключ
POLZA_API_KEY=ваш_ключ
YANDEX_IAM_TOKEN=ваш_токен
YANDEX_FOLDER_ID=ваш_folder_id
```

### 5. Запуск бота

```bash
python src/bot.py
```

## 🧩 Команды бота

### Пользовательские команды

- `/start` — начать работу
- `/help` — справка по командам
- `/stats` — статистика (только для администраторов)

### Административные команды

- `/clear` — очистить историю сообщений
- `/block <user_id>` — заблокировать пользователя
- `/unblock <user_id>` — разблокировать пользователя

### Интерфейс

- Кнопка **"Анализ ИИ"** — получить интерпретацию выбранной пословицы
- Кнопка **"Добавить"** — добавить новую пословицу
- Кнопка **"Редактировать"** — управление базой (только для администраторов)
